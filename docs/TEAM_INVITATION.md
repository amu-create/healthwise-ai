# HealthWise 프로젝트 팀 초대

안녕하세요!

AI 기반 헬스케어 플랫폼 HealthWise 프로젝트에 참여하게 되신 것을 환영합니다! 🎉

## 🚀 시작하기

### 1. GitHub 저장소
- **URL**: https://github.com/amu-create/healthwise-ai
- **접근 권한**: GitHub 사용자명을 알려주시면 초대하겠습니다

### 2. 개발 환경 설정 (10분 소요)
```bash
# 1. 프로젝트 클론
git clone https://github.com/amu-create/healthwise-ai.git
cd healthwise-ai

# 2. Docker로 실행
docker-compose -f docker-compose.dev.yml up

# 3. 브라우저에서 확인
http://localhost:3000
```

### 3. 필요한 도구
- **필수**: Docker Desktop, Git
- **권장**: VS Code, GitHub Desktop

### 4. API 키 (별도 전달)
다음 서비스의 API 키가 필요합니다:
- OpenAI API
- Google Gemini API
- YouTube API
- Kakao Maps API
