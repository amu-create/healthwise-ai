## 운동 영상 페이지 문제 해결

### 문제 상황
1. 음악 추천 페이지에서 운동과 관련 없는 음악 영상들이 표시됨
2. YouTube 썸네일 404 에러 발생
3. `/api/music/youtube-search/` API가 500 에러 반환

### 원인
1. 음악 추천 페이지에서 YouTube API 호출 시 음악 카테고리(10)로 검색
2. 운동 관련 키워드가 아닌 일반 음악 검색
3. 일부 비디오 ID가 유효하지 않거나 권한 문제로 썸네일 접근 불가

### 해결 방안
1. 운동 영상과 음악 추천을 명확히 분리
2. 음악 추천 페이지에서는 운동용 음악/플레이리스트만 검색하도록 쿼리 수정
3. 썸네일 로드 실패 시 대체 이미지 사용
4. YouTube API 키 설정 확인

### 수정 필요 파일
- `backend/apps/api/music/views.py` - YouTube 검색 쿼리 개선
- `frontend/src/pages/Music/index.tsx` - 이미지 오류 처리 추가
